# Bridge-led-lighting-with-push-button-22-04-2026
component used :
arduino uno , breadboard ,leds(3),push button,jumper wires .

how it works:
1) What Is This Project?
This is a simple LED control project built using Arduino Uno. The player presses a push button to turn 3 LEDs ON and OFF at the same time with each click!

2) Purpose:
To control 3 LEDs simultaneously using a single push button — one click turns all ON, next click turns all OFF!
